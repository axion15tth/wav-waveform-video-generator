# WAV波形動画ジェネレーター

WAVファイルから美しい波形アニメーション動画を生成するWebアプリケーションです。

## 特徴

- 🎵 WAVファイルのドラッグ&ドロップアップロード
- 🎨 カスタマイズ可能な波形と背景の色
- 📹 複数の動画解像度（HD、Full HD、4K）
- ⚡ 複数のフレームレート（24、30、60 fps）
- 💾 WebM形式で動画をエクスポート

## 使い方

1. Webアプリにアクセス
2. WAVファイルをドラッグ&ドロップまたは選択
3. 波形の色、背景色、動画サイズ、FPSをカスタマイズ
4. 「動画を生成」ボタンをクリック
5. 生成完了後、動画をダウンロード

## 技術スタック

- **Web Audio API** - 音声データの解析と処理
- **Canvas API** - 波形のリアルタイム描画
- **MediaRecorder API** - 動画のキャプチャとエンコード
- **Pure JavaScript** - フレームワーク不要のシンプルな実装

## ブラウザ互換性

Chrome、Edge、Operaなどの最新のモダンブラウザで動作します。
Safari、Firefoxは一部機能に制限がある場合があります。

## ローカルでの実行

リポジトリをクローンして、`index.html`をブラウザで開くだけで動作します。
ビルドプロセスや依存関係のインストールは不要です。

```bash
git clone <repository-url>
cd muramatsu
# index.htmlをブラウザで開く
```
